Markdown Guide 🖊

Markdown is a way to use characters on a keyboard to format text like HTML elements! Here is a quick guide to Markdown

# Header 1

## Header 2

### Header 3

#### Header 4

##### Header 5
###### Header 6

You can make elements **bold** by adding the double stars and make elements *italic* by adding 1 star on each side

For a more comprehensive gudie to Markdown syntax, refer to the following link: Markdown Syntax Guide
